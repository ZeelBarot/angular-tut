Interpolation in Angular

  • Interpolation works only within a component

  • Interpolation evaluates expressions inside {{signI}}

Limitations:

  • Cannot assign or create new variables

  • Cannot use increment / decrement operators

  • Cannot use loops or conditions in normal JS form

{{ name }}

{{ 2 + 2 }}

{{ x + y }}

{{ 10 == 10 }}

{{ x == y }}

'hello'.toUpperCase() => {{ 'hello'.toUpperCase() }}
