## Topics Covered

### 1. What Angular Is
- Angular is a framework used to build **large, structured web applications**
- Follows **Single Page Application (SPA)** architecture
- UI updates without full page reload
- Maintained and developed by **Google**

---

### 2. Angular Components
- A **component** is a small, independent part of the UI
- Each component controls its own:
  - Logic (`.ts`)
  - Template / UI (`.html`)
  - Styles (`.css`)
- Components help in:
  - Code reusability
  - Separation of concerns
  - Maintainability

---

### 3. Angular Data Binding
Learned how TypeScript and HTML communicate:

- **Interpolation (`{{ }}`)** ‚Üí display data
- **Property Binding (`[ ]`)** ‚Üí bind element properties
- **Event Binding (`( )`)** ‚Üí handle user actions
- **Two-Way Binding (`[( )]`)** ‚Üí sync input and data

Key idea:
- Data flow is controlled and predictable

---

### 4. Angular Directives
Directives tell Angular **how to change the DOM**.

#### Types of Directives:
- **Attribute Directives**
  - Modify appearance or behavior
  - Examples: `ngClass`, `ngStyle`, `ngModel`
- **Structural Directives**
  - Create or remove DOM elements
  - Examples: `*ngIf`, `*ngFor`, `*ngSwitch`

Key difference:
- Attribute ‚Üí modifies existing elements
- Structural ‚Üí adds/removes elements from DOM
